# Czwiczenie3

# Konsolowa aplikacja do analizy danych uczelni z wykorzystaniem LINQ.

# zad1 - Metoda: Studenci z Warszawy, operator LINQ: Where
# zad2 - Metoda: Adresy e-mail studentów, operator LINQ: Select
# zad3 - Metoda: Studenci posortowani alfabetycznie, operator LINQ: OrderBy, ThenBy
# zad4 - Metoda: Pierwszy przedmiot z kategorii Analytics, operator LINQ: FirstOrDefault
# zad5 - Metoda: Czy istnieje nieaktywny zapis, operator LINQ: Any
# zad6 - Metoda: Czy wszyscy prowadzący mają katedrę, operator LINQ: All
# zad7 - Metoda: Liczba aktywnych zapisów, operator LINQ: Count
# zad8 - Metoda: Unikalne miasta studentów, operator LINQ: Distinct
# zad9 - Metoda: Trzy najnowsze zapisy, operator LINQ: OrderByDescending, Take
# zad10 - Metoda: Druga strona przedmiotów (paginacja), operator LINQ: Skip, Take
# zad11 - Metoda: Połączenie studentów z zapisami, operator LINQ: Join
# zad12 - Metoda: Pary student–przedmiot, operator LINQ: SelectMany, Join
# zad13 - Metoda: Grupowanie zapisów według przedmiotu, operator LINQ: GroupBy
# zad14 - Metoda: Średnia ocena na przedmiot, operator LINQ: Average
# zad15 - Metoda: Prowadzący i liczba przedmiotów, operator LINQ: GroupBy, Count
# zad16 - Metoda: Najwyższa ocena każdego studenta, operator LINQ: Max
